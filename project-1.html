Stock Price
Prediction

Stock price prediction using with an ETS (Error, Trend and Season) model.

I was interested in exploring the use of time series analysis for predicting stock prices, so I decided to undertake a project that involved using web scraping and the ETS (Error, Trend, Seasonality) model to predict the future stock prices of Exxon Mobil Corporation (XOM).

To begin, I set up an account with the Federal Reserve Economic Data (FRED) website and obtained an API key, which allowed me to access their vast collection of economic data. I then used Python to develop a web scraping script that accessed the FRED API and extracted the necessary data for the project, including XOM prices, WTI prices, and macroeconomic data.

Once I had collected the data, I used the ETS model to analyze the time series and make predictions about future XOM stock prices. I tested both a triple additive and a triple multiplicative ETS model to see which provided the best results. However, it is important to note that even the most sophisticated models cannot guarantee perfect accuracy when it comes to predicting stock prices.

Overall, the project highlighted the potential benefits of using web scraping and time series analysis for financial decision-making, and demonstrated the power of the ETS model for predicting stock prices. While it is important to approach financial decision-making with caution, the use of these tools and techniques can provide valuable insights and help to inform investment strategies.
