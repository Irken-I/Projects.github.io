Bee Game

Final project for CMU computer science class

This game uses object-oriented programming (OOP) principles, with separate classes for flowers, bees, and players. The flower class contains methods for growing flowers when the player is close to them, and for adding or removing flowers from the list.

To manage the pollen inventory, a list is used, and the oldest pollen is removed when the count gets above 6. The bee's movement is gradual and depends on the distance between the bee and the mouse. The animated sprite of the bee is updated to show the wings flapping faster when the bee is flying faster.

The flowers grow smoothly when the bee is nearby and holding the right pollen. The game keeps track of which pollen indicators or pollinating flowers should grow, and this is achieved by assigning each flower object a unique integer ID. The pollen inventory is a list containing flower IDs. The flowers have a sinusoidal wobble effect created by varying the x offset of each flower using math.
